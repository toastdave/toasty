# Ratings And Checklists

## Future loop

After discovery proves out, users should be able to log anime and rate it with richer signals than a single score.

## Rating system shape

- rubric-driven by media type
- weighted multi-axis scoring
- spoiler-aware text review support
- vibe badges for playful identity

## Anime starter axes

- craft
- fun
- emotional impact
- rewatchability
- pacing
- spectacle
- weirdness
- comfort
- recommendation strength

## Checklist states

- planned
- in progress
- done
- abandoned
