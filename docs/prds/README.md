# PRD Index

This directory keeps the long-range planning docs for Toasty. The app starts anime-first, but every doc assumes a future expansion path into movies, TV, books, and music once the core discovery loop is stable.

## Document map

- `00-product-vision.md` - product direction, experience pillars, and MVP framing
- `01-platform-foundation.md` - stack, monorepo structure, environments, and guardrails
- `02-anime-catalog-and-ingestion.md` - canonical media model and Jikan-first ingestion plan
- `03-discovery-and-trending.md` - top charts, seasonal schedules, filters, and schedule progress
- `04-ratings-and-checklists.md` - rubric-driven ratings, vibe badges, and progress tracking
- `05-lists-profiles-and-social.md` - profile shape, lists, and social-lite roadmap
- `06-tournaments-and-year-end-mode.md` - yearly ranking, seeding, bracket flow, and voting
- `implementation-roadmap.md` - phased delivery plan
