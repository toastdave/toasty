# PRD Index

This directory keeps the long-range planning docs for Toasty. The app launches anime-first, but the roadmap now assumes a broader screen-media future across anime, TV shows, and movies once the discovery and rating loops are strong enough to support recommendations and annual tournament play.

## Document map

- `00-product-vision.md` - product direction, experience pillars, cross-media ambition, and interface posture
- `01-platform-foundation.md` - stack, monorepo structure, environments, and guardrails
- `02-anime-catalog-and-ingestion.md` - canonical media model, Jikan-first ingestion, and future adapter path
- `03-discovery-and-trending.md` - browse surfaces, charts, schedules, and recommendation entry points
- `04-ratings-and-checklists.md` - single-page rating canvas, universal core dimensions, flavor vectors, and tracking
- `05-lists-profiles-and-social.md` - profile shape, public identity, lists, activity, and social-lite roadmap
- `06-tournaments-and-year-end-mode.md` - annual tournament setup, seeding rules, bracket flow, and archives
- `07-current-state-and-next-sprint.md` - snapshot of shipped work and the immediate strategic priorities
- `08-recommendations-and-intelligence.md` - recommendation engine direction, explainability rules, and rollout stages
- `implementation-roadmap.md` - phased delivery plan across discovery, ratings, recommendations, multi-media expansion, and tournaments
